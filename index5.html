  • Extension recommendation

Material Theme

This extension is really useful extension to change the background color(theme) of your coding program. There are several choices you can choose. Just pick any theme you want to use.

Live Server

This extension is really useful extension to see what you made. There will be new button at the right bottom. when you press that button, it will directly go to your coding website.

Material Icon Theme

This extension is really useful extension to change the theme of your coding icon such as HTML, JS and Css. Those icons will make your coding atmosphere looks better.

Bracket Pair Colorizer

This extension is really useful extension to intesify the prettier atmosphere. Originally, the brackets have no color. It will be just grey color. However, after you use this extension, it will make your coding prettier.

Auto Rename Tag

This extension is really useful extension for whole coding process. It directly fix all the errors by itself. It can fix the words in bracket at the same time.

ESLint & jshint

This extension is really useful extension to fix all the errors. As it says in its name, it keeps hinting us the way to make better. It's kind of personal teacher for coding.
